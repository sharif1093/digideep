Welcome to Digideep's documentation!
====================================

Digideep is a pipeline for fast prototyping Deep Reinforcement Learning (DeepRL) algorithms
which uses `PyTorch <https://github.com/pytorch/pytorch>`_ and `Gym <https://github.com/openai/gym>`_ / `dm_control <https://github.com/deepmind/dm_control>`_.
